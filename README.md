An intelligent agent-based system designed to reduce post-harvest losses and improve food security for smallholder farmers in Uganda.

#How it works:

#Real-time Risk Monitoring#: Tracks 184 farms across multiple districts (Mubende, Masaka, Kapchorwa) with color-coded risk levels (High/Medium/Low).

#AI-Powered Recommendations#: Provides personalized advice like "Sell within 48 hours" or "Apply solar drying today" based on humidity and market conditions.

#Live Weather Integration#: Displays current weather impact with humidity alerts and rain predictions.

#Market Price Trends#: Visualizes Kampala market prices for matooke and other staples to help farmers time their sales.

#Chat Agent Interface#: Farmers can ask questions and receive instant AI guidance.

#SMS Simulation#: Demonstrates how remote farmers receive alerts via mobile phones.
